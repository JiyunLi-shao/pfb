Related Doc: package pfb

class PFBLane[T <: Data, V] extends Module

A single lane of the PFB. A full PFB will include >= 1 lanes operating in parallel. The PFB consists of delay parallel FIR filters, which means taps.length / delay windows are added together. As a result, delay is required to divide taps.length evenly.

The FIR filter is implemented in transposed form. Delays are implemented with chisel Mems which may be implemented with SRAMs.
